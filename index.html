💡 기본 기능

  • ✅ 체크리스트 관리: 준비물을 선택하면 진행률 상태링과 진행률 백분율이 실시간 업데이트됩니다.
  • 💾 자동 저장: 체크 상태, 정렬 상태, 배너 등 모든 변경사항은 브라우저에 즉시 안전하게 자동 저장됩니다.

✏️ 편집 모드 (커스터마이징)

  • ↕️ 항목 위치 변경 (드래그): ✏️ 편집 모드를 누르고 좌측 손잡이(⠿)나 항목 자체를 마우스/손가락으로 드래그하여 순서를 마음대로 바꿀 수 있습니다.
  • 📷 나만의 배너 업로드: 편집 모드 상태에서 상단 배너 영역의 📷 배너 이미지 변경 단추를 누르면 원하는 사진으로 커스텀 배너를 지정할 수 있습니다.

📸 이미지 내보내기 & 리셋

  • 📸 이미지로 저장 (JPG): 📸 이미지로 저장 단추를 누르면 내가 체크한 항목과 진행률(선택 항목 비율)이 담긴 고해상도 JPG 이미지 카드를 다운로드할 수 있어, 오프라인이나 락페 현장에서 사진첩으로 빠르게 볼 수 있습니다.
  • 🔄 초기화: 리스트 상태를 완전히 처음 기본값으로 되돌리고 싶을 때 사용합니다.

📸 저장용 이미지 미리보기

📱 아이폰/안드로이드 또는 카카오톡/인스타그램 웹뷰 환경에서는
아래 이미지를 길게 꾹 누른 뒤 [사진 앱에 추가] 또는 [이미지 저장]을 눌러 갤러리에 저장해 주세요!

🆕 업데이트 내역 (Changelog)

v1.3.0 2026-07-17 Latest
Added
  • iOS 및 모바일 사진첩 직접 저장 기능(Web Share API): 이미지 저장 시 공유창이 바로 팝업되어 네이티브 사진첩 저장 지원. 카카오톡/인스타그램 웹뷰 전용 이중 이미지 보기 꾹누르기 저장 모달 백업 탑재.
  • 체크리스트 사용 설명 가이드 탑재: 기본 체크, 커스터마이징 드래그, 이미지 저장 설명 등이 들어있는 도움말 팝업 연동.
  • iOS 홈 화면 추가 최적화: 홈 화면에 추가할 때 표시되는 전용 앱 아이콘 제작 및 잘림 없는 '후지록' 전용 이름 반영.
  • 해외여행 필수품 리스트 추가: 여권, 항공권, 여행자보험, 일본 교통카드, eSIM, 엔화 현금 항목 개설.
  • 기타 유용한 준비물 보강: 물티슈, 쿨링티슈, 이어플러그, 방수팩, 지퍼백, 벌레약, 립밤, 상처연고, 방수테이프 추가.
  • 체크리스트 캐시 자동 병합: 준비물 대량 추가/이동 시 기존 사용자 브라우저 체크 상태를 안전하게 병합하는 마이그레이터 반영.
Changed
  • 카테고리 구성 직관화: 준비물 항목들을 '[공통]', '[숙소 이용자용] 캐리어', '[캠핑/차박용]' 카테고리로 재구획.
  • 레이아웃 시인성 및 반응형 글자 크기 보완: 배너 글씨 뒤 그림자 및 오버레이 강화, 모바일 크기 반응형 clamp 글자 조절.
  • 준비물 명칭 변경 및 위치 개선: 타임테이블 → 후지락앱, 햇 (모자) → 모자, 플리스 → 바람막이, 판초 우의(파랑) → 판초 우의 개칭 및 캐시 자동 매핑. 맥주 너클 오프너를 배낭 하위로 이동, 중요 소지품을 당일 챙길 것 카테고리로 상향.
  • 스크롤 및 외부 에러 팝업 제거: 모달 오픈 시 배경 휠 잠금, 닫기 버튼 추가. 타사 광고 차단 앱 등 외부 스크립트 오작동 시 알림창이 뜨지 않도록 처리.
  • 정렬 시 체크 상태 유지: 준비물 드래그 앤 드롭 이동 시, 선택된 체크박스 상태가 초기화되지 않고 동기화 저장되도록 개선.
Removed
  • 불필요한 중복 소지품 항목 제거 (건강보험증, 집 열쇠, 페이스 시트, 해외결제 가능 신용카드 항목 등).
